# Develop & Build Requirements

- NodeJS LTS
- pnpm

# Setup

```bash
git clone https://github.com/kerthical/chatgpt
cd chatgpt
pnpm install
```

# Debug

```bash
pnpm dev
```

# Build

The dist file is only `./dist/index.html`.

```bash
pnpm build
```
